## Installation

* Download the code zip file or clone this repository
* Run npm install in the directory and dependicies will installed
* Run npm start to run the app
* app will run http://localhost:3000/

## Libraries used

* Styled Component
* Redux
* React Router
* Firebase
* slick-carousel
